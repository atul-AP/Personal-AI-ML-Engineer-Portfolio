# 🎨 AI/ML Engineer Portfolio Design (Figma)

## 📌 Project Overview
This project is a UI/UX design of a modern personal portfolio website created using Figma. The portfolio is designed to showcase skills, projects, education, and professional experience of an AI/ML Engineer.
